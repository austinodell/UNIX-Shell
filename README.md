# UNIX Shell

## A Semester Long Project

Villanova University  
CSC 1600: Operating Systems

Pair programmed by Austin Odell and Brian Dicola.

The shell supports:  
  -  Command Line Parsing
  -  Process Execution
  -  I/O Redirection
  -  Background Processing
  -  Signal Handling
     -  SIGINT (ctrl-c)
     -  SIGTSTP (ctrl-z)
  -  Resume
     -  SIGCONT (enter 'resume')
