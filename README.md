# UNIX Shell

## A Semester Long Project

Villanova University  
CSC 1600: Operating Systems

Pair programmed by Austin Odell and Brian Dicola.

The shell supports:  
1.  Command Line Parsing
2.  Process Execution
3.  I/O Redirection
4.  Background Processing
5.  Signal Handling
  1.  SIGINT (ctrl-c)
  2.  SIGTSTP (ctrl-z)
6.  Resume
  1.  SIGCONT (enter 'resume')
